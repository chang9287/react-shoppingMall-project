## 📌 페이지 설명 
|                               메인 페이지                               |                               서브 페이지                               |

|    대표기능 : 슬라이더(main Visual, banner), 아코디언, Tab 안에 Tab기능    |  공지사항 페이지 : 공지사항을 한 눈에 확인할 수 있으며, 효율적으로 탐색할 수 있도록 페이지네이션으로 정렬                                         |
